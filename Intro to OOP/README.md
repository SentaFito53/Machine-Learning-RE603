# 📊 Exploratory Data Analysis (EDA) Project

## 📝 Deskripsi
Repository ini berisi project **Exploratory Data Analysis (EDA)**.
Project ini mencakup tiga proses utama dalam persiapan data, yaitu **Outlier Handling**,
**Missing Value Handling**, dan **Data Encoding**.

---

## ⚙️ Proses yang Dilakukan

### 1. 🔍 Outlier Handling
Proses identifikasi dan penanganan nilai ekstrem dalam dataset menggunakan metode
IQR dan Z-Score agar distribusi data lebih normal.

### 2. 🧹 Missing Value Handling
Proses penanganan nilai yang hilang atau kosong dalam dataset menggunakan metode
imputasi (mean, median, modus) maupun penghapusan baris/kolom.

### 3. 🔢 Data Encoding
Proses konversi data kategorikal menjadi format numerik menggunakan
Label Encoding dan One-Hot Encoding agar data siap digunakan untuk analisis lebih lanjut.

---

## 🛠️ Library yang Digunakan
- `pandas` & `numpy` — Manipulasi data
- `matplotlib` & `seaborn` — Visualisasi data
- `scikit-learn` — Preprocessing data
