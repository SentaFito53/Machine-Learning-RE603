# 🐍 Intro to Python

## 📝 Deskripsi
Repository ini berisi materi pengenalan dasar pemrograman Python yang mencakup
**Tipe Data**, **Function**, dan **Object-Oriented Programming (OOP)**.

---

## 📚 Materi yang Dipelajari

### 1. 📦 Tipe Data
Pengenalan berbagai tipe data dasar dalam Python seperti:
- **String** — Tipe data teks
- **Integer & Float** — Tipe data angka
- **Boolean** — Tipe data benar/salah (`True`/`False`)
- **List, Tuple, Set** — Tipe data koleksi
- **Dictionary** — Tipe data pasangan key-value

### 2. ⚙️ Function
Pengenalan cara mendefinisikan dan menggunakan fungsi dalam Python, meliputi:
- Fungsi dengan dan tanpa parameter
- Fungsi dengan return value
- Lambda function
- Scope variabel (local & global)

### 3. 🏗️ Object-Oriented Programming (OOP)
Pengenalan konsep pemrograman berorientasi objek dalam Python, meliputi:
- **Class & Object** — Cara membuat dan menggunakan class
- **Attribute & Method** — Properti dan perilaku dari sebuah class
- **Inheritance** — Pewarisan sifat antar class
- **Encapsulation** — Pembatasan akses terhadap data
